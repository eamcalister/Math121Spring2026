    Section Applications of the Angle Sum Identities

    In class yesterday we did an activity to derive the first of the identities in the following:

    The Angle Sum Identities.

    If \(\theta\) and \(\phi\) are angles, then we have
    \begin{align*} \sin(\theta+\phi) \amp = \sin(\theta)\cos(\phi) + \cos(\theta)\sin(\phi) \\ \sin(\theta-\phi) \amp = \sin(\theta)\cos(\phi) - \cos(\theta)\sin(\phi) \\ \cos(\theta+\phi)\amp = \cos(\theta)\cos(\phi) - \sin(\theta)\sin(\phi) \\ \cos(\theta-\phi)\amp = \cos(\theta)\cos(\phi) + \sin(\theta)\sin(\phi) \end{align*}

    Subsection Sums of Sinusoids with the Same Period

    In the mathematical description of oscillating objects, one frequently encounters functions of the form
    \begin{equation*} f(t) = c_1\cos(Bt) + c_2\sin(Bt), \end{equation*}
    where \(c_1\) and \(c_2\) are constants. This is a periodic function with period \(P = \frac{2\pi}{B}\text{,}\) but we would like to know its amplitude too. To do so, we can write it in the form
    \begin{equation*} f(t) = c_1\cos(Bt) + c_2\sin(Bt) = A\sin(Bt + \phi) \end{equation*}
    for some constants \(A\) and \(\phi\text{.}\) To see how, letâ€™s do an example.
    ðŸ”—

    Example 1.

    Let \(f(t) = 3\cos(7t) + 2\sin(7t)\text{.}\) To write in the form \(f(t) = A\sin(7t + \phi)\text{,}\) we expand the given form using the angle sum identity for sine.
    \begin{align*} 3\cos(7t) + 2\sin(7t) \amp = A\sin(7t + \phi)\\ \amp = A\cos(7t)\sin(\phi) + A\sin(7t)\cos(\phi). \end{align*}
    Equating coefficients of \(\cos(7t)\) and \(\sin(7t)\text{,}\) we have
    \begin{align*} 3 \amp = A\sin(\phi)\\ 2 \amp = A\cos(\phi) \end{align*}
    Thus,
    \begin{equation*} \tan(\phi) = \frac{A\sin(\phi)}{A\cos(\phi)} = \frac{3}{2}. \end{equation*}
    Then we can take \(\phi = \tan^{-1}(1.5)\approx 0.983\text{.}\) Also,
    \begin{equation*} A^2 = A^2\sin^2(\phi) + A^2\cos(\phi) = 9 + 4. \end{equation*}
    So, we \(A = \sqrt{13}\text{.}\) Hence,
    \begin{equation*} f(t) = 3\cos(7t) + 2\sin(7t) = \sqrt{13}\sin(7t + \tan^{-1}(1.5)). \end{equation*}

    Subsection Sum to Product Identities and Applications

    Some identities related to the angle sum identities are the following:

    Sum to Product Identities.

    If \(\theta\) and \(\phi\) are angles, then we have
    \begin{align*} \sin(a) + \sin(b) \amp = 2\sin\left(\frac{a+b}{2}\right)\cos\left(\frac{a-b}{2}\right) \\ \sin(a) - \sin(b) \amp = 2\cos\left(\frac{a+b}{2}\right)\sin\left(\frac{a-b}{2}\right) \\ \cos(a) + \cos(b) \amp = 2\cos\left(\frac{a+b}{2}\right)\cos\left(\frac{a-b}{2}\right) \\ \cos(a) - \cos(b) \amp = -2\sin\left(\frac{a+b}{2}\right)\sin\left(\frac{a-b}{2}\right) \end{align*}

    Example 2.

    Suppose we want to solve
    \begin{equation*} \cos(4x) + \cos(x) = 0,\ \ 0\leq x \leq \pi. \end{equation*}
    We apply the third identity given above to re-write the equation as
    \begin{equation*} 2\cos\left(\frac{5x}{2}\right)\cos\left(\frac{3x}{2}\right) = 0. \end{equation*}
    This is a factored form, so we can set \(\cos\left(\frac{5x}{2}\right)\) and \(\cos\left(\frac{3x}{2}\right)\) equal to zero separately on the given domain to obtain
    \begin{equation*} x = \frac{\pi}{5},\ \frac{3\pi}{5},\ \pi,\ \frac{\pi}{3}. \end{equation*}
