    Definition of \(\lim\limits_{x\to a} f(x)\).

    If we are given a function \(f\) and a number \(a, \) we say that \(\displaystyle \lim_{x \to a} f(x) = L \) if we can make \(f(x) \) as close to \(L\) as we would like by taking values of \(x\) sufficiently close to, but not equal to, \(a, \) i.e. \(L \) is the value \(f(x) \) approaches as \(x \) approaches (but does not equal) \(a .\)
