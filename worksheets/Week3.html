    Recall that in Week 1 Activity - ExerciseÂ 3 we found a formula for the exponential function passing through the points \((1,4)\) and \((5,12)\) by solving a system of equations. The formula was
    \begin{equation*} f(x) = \frac{4}{\sqrt[4]{3}}(\sqrt[4]{3})^x. \end{equation*}
    We can build that same function starting with the base function
    \begin{equation*} g(x) = 4(3)^x. \end{equation*}
    We chose this function because \(4\) is the \(y\)-value of the initial point (\((1,4)\)) and the \(3\) is the ratio of the \(y\)-values of the later point over the earlier point (\(12/4\)). The graph of \(g(x)\) passes through the points \((0,4)\) and \((1,12)\text{,}\) and we can apply transformations so that the resulting function passes through the given points \((1,4)\) and \((5,12)\text{.}\)

    Once you can find a formula for an exponential function that looks like

    \begin{equation*} f(t) = a e^{{\frac{k(t-h)}{P}}}. \end{equation*}
